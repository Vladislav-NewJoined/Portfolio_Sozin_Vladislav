# ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ ПРОЕКТА LaravelDemo

## Запуск проекта

1. Откройте терминал в корневой директории проекта
2. Выполните команду для запуска сервера разработки:
   php artisan serve
3. Сервер будет запущен по адресу http://localhost:8000 или http://127.0.0.1:8000

## Доступные страницы

1. Главная страница Laravel:
   http://localhost:8000

2. Страница приветствия:
   http://localhost:8000/hello
   Отображает приветственное сообщение и список возможностей Laravel.

3. Страница с формой:
   http://localhost:8000/form
   Позволяет ввести имя и получить персонализированное приветствие.

## Структура проекта

- app/Http/Controllers/HelloController.php - контроллер с методами для обработки запросов
- resources/views/hello.blade.php - шаблон для страницы приветствия
- resources/views/form.blade.php - шаблон для страницы с формой
- routes/web.php - файл с маршрутами приложения

## Основные команды

- php artisan serve - запуск сервера разработки
- php artisan make:controller ИмяКонтроллера - создание нового контроллера
- php artisan route:list - просмотр всех зарегистрированных маршрутов
- php artisan config:clear - очистка кеша конфигурации

## Требования

- PHP 8.1 или выше
- Composer
- Расширения PHP: fileinfo, pdo_sqlite, sqlite3

## Установка зависимостей

Если вы клонировали проект из репозитория, выполните:
composer install

## Настройка окружения

1. Скопируйте файл .env.example в .env:
   cp .env.example .env
2. Сгенерируйте ключ приложения:
   php artisan key:generate
3. Настройте подключение к базе данных в файле .env
