ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ ПРИЛОЖЕНИЯ
Запуск приложения
1.	Откройте проект в IntelliJ IDEA.
2.	Запустите класс Main (щелкните правой кнопкой мыши по файлу Main.java и выберите "Run 'Main'").
3.	Дождитесь появления в консоли лога: "Started Main in 11.136 seconds (process running for 19.364)".
4.	Перейдите на веб-страницу по адресу: http://localhost:8080/, на которой реализован функционал проекта.
Работа с приложением
Выбор элементов
•	Выбор одного элемента: Нажмите на чекбокс (галочку) слева от элемента, чтобы выбрать его. Выбранные элементы будут выделены цветом.
•	Выбор нескольких элементов: Вы можете выбрать несколько элементов, просто нажимая на чекбоксы нужных элементов.
•	Отмена выбора: Чтобы отменить выбор элемента, нажмите на чекбокс выбранного элемента еще раз.
Поиск и фильтрация
•	Поиск элементов: Введите текст в поле поиска в верхней части страницы. Результаты будут отображаться автоматически по мере ввода.
•	Очистка поиска: Чтобы очистить поиск и вернуться к полному списку, удалите текст из поля поиска.
•	Пагинация результатов поиска: При поиске отображается не более 20 элементов. Прокрутите страницу вниз, чтобы подгрузить следующие 20 элементов, соответствующих критериям поиска.
Сортировка элементов (Drag&Drop)
•	Изменение порядка элементов: Чтобы изменить порядок элементов, наведите курсор на элемент, который хотите переместить. Нажмите и удерживайте левую кнопку мыши на значке "☰" слева от элемента.
•	Перетаскивание: Не отпуская кнопку мыши, перетащите элемент в нужное место в списке. Во время перетаскивания вы увидите индикатор, показывающий, куда будет помещен элемент.
•	Завершение перетаскивания: Отпустите кнопку мыши, когда элемент находится в нужной позиции. Порядок элементов будет обновлен автоматически.
•	Перетаскивание нескольких элементов: Если вы выбрали несколько элементов (с помощью чекбоксов) и начинаете перетаскивать один из выбранных элементов, все выбранные элементы будут перемещены вместе, сохраняя свой относительный порядок.
Сортировка отфильтрованных результатов
•	Сортировка результатов поиска: Вы можете изменять порядок элементов в результатах поиска так же, как и в полном списке, используя Drag&Drop.
•	Сохранение порядка: Изменения порядка в результатах поиска будут применены к общему списку элементов.
Пагинация и подгрузка элементов
•	Начальная загрузка: При первом открытии страницы загружается только первые 20 элементов.
•	Подгрузка при прокрутке: Прокрутите страницу вниз до конца текущего списка, чтобы автоматически подгрузить следующие 20 элементов.
•	Индикатор загрузки: Во время подгрузки новых элементов внизу страницы будет отображаться индикатор загрузки.
Сохранение состояния при перезагрузке
•	Сохранение выбранных элементов: При перезагрузке страницы все ранее выбранные элементы останутся выбранными.
•	Сохранение порядка элементов: Порядок элементов, который вы установили с помощью Drag&Drop, сохраняется при перезагрузке страницы.
•	Пагинация после перезагрузки: После перезагрузки страницы будут отображаться первые 20 элементов в установленном вами порядке. Остальные элементы будут подгружаться при прокрутке.
Сброс порядка элементов
•	Кнопка сброса: Если вы хотите вернуть элементы к исходному порядку, нажмите кнопку "Сбросить порядок" в верхней части страницы.
•	Подтверждение сброса: После нажатия на кнопку порядок элементов будет сброшен к исходному, но выбранные элементы останутся выбранными.
Дополнительная информация
•	Все изменения (выбор элементов, изменение порядка) автоматически сохраняются на сервере.
•	Приложение работает в режиме реального времени, все изменения применяются мгновенно.
•	Если у вас возникли проблемы с отображением или функциональностью, попробуйте обновить страницу (F5).
